## English

### ✨ Features

- ⚡ **Lightning Fast** - Built with FastAPI and React for maximum performance
- 🔒 **Secure** - JWT authentication, bcrypt hashing, OWASP best practices
- 🎨 **Beautiful UI** - Minimalist design with Tailwind CSS
- 📊 **Observable** - Complete monitoring stack (Prometheus, Grafana, Loki)
- 🐳 **Docker Ready** - One command deployment
- 🧪 **Well Tested** - Comprehensive test coverage with Pytest
- 📱 **Responsive** - Works perfectly on all devices
- 🔄 **Real-time** - WebSocket support for live updates

### 🛠️ Tech Stack

**Frontend:**
- React 18 + TypeScript
- Tailwind CSS
- Redux Toolkit
- Socket.io

**Backend:**
- FastAPI (Python 3.11+)
- SQLAlchemy ORM
- PostgreSQL
- Redis (caching)
- Celery (async tasks)

**DevOps:**
- Docker & Docker Compose
- GitHub Actions (CI/CD)
- Prometheus + Grafana
- Sentry (error tracking)

### 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/koljaPl/fastapi-blog.git
cd blog-platform

# Start everything with Docker
docker-compose up -d

# Access the application
# Frontend: http://localhost:3000
# Backend API: http://localhost:8000/docs
```
